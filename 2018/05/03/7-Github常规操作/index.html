7.本地代码上传Github

Posted on 2018-05-03 | In Github入门 | Comments: | Views:

本地代码上传及报错git fatal: 拒绝合并无关历史

第一步: 建立git仓库

cd你要上传的项目根目录

第二步: 将项目中的文件add到仓库缓存

特定某个文件add 文件名

第三步: 将缓存的文件提交到仓库中

第四步: 去github创建新的Repository

**点击Creat repository,进入下图界面,拷贝新建仓库的地址,红框所示

第五步: 重点!!将本地的仓库关联Github

地址用你上一步拷贝的

第六步: 上传之前,先pull,保证本地仓库已经同步远端仓库

请一定要加--allow-unrelated-histories, 否则会出现拒绝合并的错误
会出现以下信息:

在空白处直接输入任意字符, crtrl + X保存退出

第七步: 最后一步

如果没有异常, 会提示你输入你github 的Username和Password
