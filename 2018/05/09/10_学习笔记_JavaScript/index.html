    数据类型

    概述

  • ES6的6种数据类型
    • 数值(number): 整数和小数
    • 字符串(string): 文本
    • 布尔值(bool)
    • undefined: 未定义
    • null:空值
    • 对象(object): 各类值的集合,理解为go中结构体的对象
      • 狭义对象(object)
      • 数组(array)
      • 函数(function)->数据类型
        tips: var a = 函数
  • type of 运算符,返回数据类型
    • 用来检查外来变量
    • type of null为object,历史遗留问题

      null,undefined和布尔值

  • 辨析null和undefined,语法效果相同,都表示没有

    • if(!undefined)和if(!null),;条件成立, 此时undefined和null表示false
    • null表示空和没有,undefined表示没定义

  • Bool六个被转为False的值

    • undefined
    • null
    • false
    • 0
    • NaN
    • “”或者’’空字符串

      概述

  • 整形和浮点型
    • 内部以64浮点型储存,1===1.0
    • 浮点型天生不精确,0.1+0.2===0.3 为false
    • 精度有限

数值表书法
